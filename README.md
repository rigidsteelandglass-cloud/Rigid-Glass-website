# Noah's Showers

Portfolio website for Noah's custom shower & bathroom remodeling work, with a
self-serve photo dashboard so Noah can add new projects without touching code.

**Astro + Tailwind · Sveltia CMS · hosted free on Netlify.**

## Quick start

```bash
npm install
npm run dev        # http://localhost:4321
```

## Editing

- **Business info** (name, phone, email, area): `src/config.ts`
- **Projects/photos**: the `/admin` dashboard (or `src/content/projects/*.md`)
- **Pages**: `src/pages/`
